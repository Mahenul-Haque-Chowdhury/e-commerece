This is a demo ecommerce storefront named "Grays" built with Next.js (App Router) and TailwindCSS. It uses local JSON for product data and Zustand for cart state.

Tech
- Next.js 15 (App Router) + TypeScript
- TailwindCSS
- Zustand for cart and overlay state
- lucide-react icons

Develop
1. Install dependencies
2. Start the dev server

Try it
```
npm install
npm run dev
```

Notes
- All images in `public/products` and `public/hero-placeholder.jpg` are placeholders. Replace with your own assets.
- The logo at `public/grays-logo.svg` is a placeholder; replace with your own brand mark.
- Data lives in `src/data/products.json`.
